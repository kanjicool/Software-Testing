# Run Testing
- Change to project directory
- Linux
``` bash
python3 -m unittest -v tests/<test_file_name.py>
```
- OSX
``` bash
python -m unittest -v tests/<test_file_name.py>
```
- Windows
``` bash
python -m unittest -v tests/<test_file_name.py>
```
# Test Coverage
- Install coverage nose2 package
``` 
pip install coverage nose2
```
- Run nose2
``` 
nose2 -v --with-coverage
```
